# 🤖 Telegram Report Bot

Telegram бот для создания отчётов о монтажных и земляных работах с экспортом в Excel и отправкой на email.

## 🌟 Возможности

- 📝 Пошаговое создание отчётов
- 📊 Экспорт в Excel/PDF
- 📧 Отправка на email
- 📈 Статистика и аналитика
- 👨‍💼 Веб-админ панель
- 📱 Мобильная совместимость
- 🔒 Безопасное хранение данных

## 🚀 Быстрый старт

### 1. Установка на локальный компьютер

```bash
# Клонирование репозитория
git clone <repository-url>
cd telegram-report-bot

# Установка зависимостей
npm install

# Настройка
npm run setup
# или вручную: cp .env.example .env

# Запуск
npm start
